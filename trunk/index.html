Homepage of pynote

Welcome to this great site, the home of pynote.

Description

Pynote is a tiny ncurses tool written in python which allows you
to create, edit and delete notes and export the data to ascii, xml or palm format.

There are not many features yet, so if you want a feature should be implemented,
please let me know and send me a mail to pynote@kai-kuehne.de.
